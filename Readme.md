C++ Blockchain Project

WORK IN PROGRESS 

This project is heavily inspired by the architecture of Bitcoin's blockchain that uses the libsecp256k library to perform modular arithmetic on 256 bit numbers.

This project is my attempt to re-create that library along witth the entire architecture in my own customized fashion. 

Note: First part of this project, is the private key generator which creates the user's identity. It has been published separately on this very github page.

Note: This requires Visual Studio 2026 with C++ installed running C++ MSVC. Stay tuned for more updates.
